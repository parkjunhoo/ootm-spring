# ootm-spring
3조 미니프로젝트 

학원에서 단기간에 짧게 진행한 프로젝트이다.<br>
첫 프로젝트고 단기간이다보니 부족한점이 많았지만, <br>
그래도 너무 즐거웠따 이게 중요한거지 ㅋㅋ<br>
다들 너무 열심히하고 잘하시는분들만 모였던거같다..<br>
진짜 짧은 기간이었지만, 서로 정도들고 많이 친해졌던 것 같다..<br>
보고싶은 우리 3조 ㅜㅜ 다들 모든일이 다 잘됬으면 좋겠다 진짜로..<br>
